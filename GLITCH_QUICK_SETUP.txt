## üÜò IF SOMETHING GOES WRONG

### QR Code Won't Appear
1. Check status endpoint in browser
2. Verify project name is correct
3. Wait 10 seconds and try again
4. Restart project: Click ‚ãÆ ‚Üí Restart Project

### WhatsApp Won't Scan
1. Check phone internet is ON
2. Check QR code is clear (not blurry)
3. Close WhatsApp completely
4. Reopen and try again
5. Generate new QR code

### Bot Not Responding
1. Check .env has correct SERVER_URL
2. Check logs: Tools ‚Üí Logs
3. Look for errors or "Connected" status
4. Restart: Click ‚ãÆ ‚Üí Restart Project
5. Re-scan QR code

### .env File Not Found
1. In Glitch, click ‚ãÆ (three dots) in sidebar
2. Click "Show hidden files"
3. .env should appear now
